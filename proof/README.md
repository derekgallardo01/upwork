# Proof — what the assets actually produce

Clients won't run Python. This folder is the **viewable** evidence: real rendered
output and console transcripts from the accelerators, captured offline. Use these as
Upwork portfolio images and as links in proposals.

> Everything here is reproducible: `python run_all.py` from the repo root regenerates
> all of it.

## How to use this in a proposal

1. Pick the asset that matches the job (the radar tells you which jobs; the asset list
   in [`../assets/README.md`](../assets/README.md) tells you which asset).
2. Paste the matching screenshot or transcript as proof — "here's this exact thing
   running."
3. Link the asset's `CASE-STUDY.md` for the problem→approach→result story.
4. Close with the offer tiers from its `OFFER.md`.
